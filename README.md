# Wireshark Traffic Analysis Lab

## Objective
Analyze network traffic using Wireshark and identify DNS and HTTP activity.

## Tools Used
- Wireshark
- Kali Linux

## Steps Performed
- Captured network traffic on interface (eth0)
- Applied DNS filter: `dns`
- Applied HTTP filter: `http`

## Findings
- Observed DNS queries and responses
- Identified HTTP GET requests
- Analyzed communication between local and external IP addresses

## Conclusion
This lab demonstrates basic network traffic analysis skills using Wireshark, including filtering and identifying common protocols like DNS and HTTP.
